Web Application Development

Course Code: ITM 1131

Course Level: Year One, Semester One

Credit Unit: 4

Course Description

This course introduces students to Web technologies - HTML, CSS, JavaScript; CSS frameworks; JavaScript frameworks. The students will learn to implement modern web pages using HTML and CSS framework. The course also introduces students to full stack web application programming using JavaScript.

Course Obectives

Upon successully completing this course, students should be able to:

  1. Dedcribe web technologies (HTML,CSS& JavaScript), CSS frsmeworks, and JavaScript frameworks
  2. Implement a simple but modern web page with HTML and CSS
  3. Build a simple but functional web application using JavaScript

Learning Outcomes

  • Use web technologies(HTML,CSS & JavaScript),CSS frameworks, and JavaScript frameworks.
  • Design and create a simple but functional web page using HTML and CSS frameworks.
  • Compose and build a simple full stack web application using JavaScript frameworks

Mode of Delivery

The mode of delivery will include, but are not limited to lecture sessions, laboratory sessions, tutorial sessions, project work.The deliveries will take face-to-face, online, and self-study mode.

Methods of Assessment

Mode of assessment will include;

  1. formative assessments:-tests, assignments, homework;
  2. summative assessments:-mid-semester examination, end of semester examination, e.g, written, viva, project report, practical;
  3. 'Authentic' or work-integrated assessment:-projects, laboratory practicals.

Relative contributions of the assessment to final grade are:

  • Formative assessments(Assignments, Homework): 10%
  • Work-integrated assessment(Projects,Laboratory practical): 20%
  • Summative assessment(2 Semester Tests):20%
  • Summative assessment(End of Semester Examination): 50%
